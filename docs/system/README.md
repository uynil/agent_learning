# Learning System Docs

这个目录只放“学习系统本身”的文件，不放具体 agent 项目的研究结论。

可以把它理解成你的研究操作系统。

## 这里放什么

- `workflow/`
  研究流程、收尾检查、主题索引
- `prompts/`
  可直接使用的 prompts、prompt 迭代记录、smoke cases
- `evaluation/`
  评估标准
- `methodology/`
  分析原则和方法
- `templates/`
  可复用的文档模板

## 这里不放什么

- 某个 agent 的架构拆解
- 某个项目的 orchestration 结论
- 某个项目里的 prompt 设计经验

这些内容应进入 [docs/research](/Users/uynil/projects/agents/docs/research)。
